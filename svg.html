SVG - Scalable Vector Graphics

SVG er et filformat for vektorgrafikk. Filene består av kunn tekst og tar derfor lite plass. Filene er også veldig gode til å skalere, fordi de ikke er pikselbaserte. Det flotte med SVG er at den kan endres med css og JavaScript.

SVG grafikk brukes ofte på nettsider, til å vise ilustrasjoner og diagrammer, logoer, animasjoner, interaktivitet, spesialefekter og å bygge interfaces og aplikasjoner.
