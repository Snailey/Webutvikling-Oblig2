GitHub

GitHub er et program, som lar grupper samarbeide om å utvikle kode. Filene blir lagret med versjon, så man kan hoppe tilbake til tidligere stadier. Og man ser hva som er endret av hvem. Det er i tillegg et backupsystem, som lagrer prosjektet i en sky, og man kan laste det ned og jobbe med det hvor man vil.
